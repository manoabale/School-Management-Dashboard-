# School-Management-Dashboard-🏫 

A responsive **School Management Dashboard** built with **HTML, CSS, and JavaScript**.  
It provides an overview of school operations including students, teachers, and classes, with interactive charts powered by **Chart.js**.



## 🚀 Features
- Sidebar navigation for quick access to sections
- Summary cards showing:
  - Total Students
  - Teachers
  - Classes
- Interactive bar chart for school statistics
- Recent activities table
- Responsive design for desktop and mobile
- Easy to extend with more charts or modules



## 🛠️ Tech Stack
- **HTML5** – structure
- **CSS3** – styling
- **JavaScript (ES6)** – interactivity
- **Chart.js** – data visualization
